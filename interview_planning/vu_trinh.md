## Things a data engineer should learn in order, based on my experience:
01. Data Modeling
02. SQL
03. Python
04. OLAP systems: cloud data warehouses, DuckDB, open table formats
05. dbt
06. Data formats: CSV, JSON, Avro, Parquet (disk), Arrow (memory)
07. Processing engine: Spark and Polars
08. Data orchestration: Airflow
09. Software engineering: Git, CI/CD, Testing, Docker, k8s (optional)
10. Message system: Kafka
11. Stream processing: Flink
12. AI-related: Basic LLM, Agent, Vector database, and how to guide AI do things for us.
